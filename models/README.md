# models/

Сюда нужно **вручную** положить файл локальной модели в формате GGUF.
Модель **не скачивается автоматически**.

Ожидается Qwen 3.5 9B в int6-квантизации. В терминах llama.cpp int6 — это
обычно `Q6_K` (см. `docs/LLM_LOCAL_INFERENCE.md`).

1. Скачайте GGUF-файл модели (например, `qwen-3.5-9b-int6.gguf` / `*-Q6_K.gguf`).
2. Положите его в эту папку: `./models/`.
3. Убедитесь, что имя файла совпадает с `basename` из `LLM_MODEL_PATH` в `.env`
   (по умолчанию `/models/qwen-3.5-9b-int6.gguf` → файл `qwen-3.5-9b-int6.gguf`).
4. Проверьте: `make check-model`.

Папка монтируется в контейнер `llm` как `/models` (только чтение модели).
